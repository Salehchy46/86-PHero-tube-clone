1. Discuss the scope of var, let, and const.

The scope of a variable in JavaScript defines where it can be accessed or used within a program. The keywords var, let, and const define variables, but they behave differently in terms of scope, hoisting, and reassignment.

1. var Scope

Function Scope: Variables declared with var are scoped to the function they are declared in. If var is declared outside of any function, it becomes global.

No Block Scope: var does not recognize block scope (e.g., within an if or for block), meaning it "leaks" out of the block and remains accessible in the outer scope.

2. let Scope

Block Scope: let variables are scoped to the block in which they are defined. They cannot be accessed outside the block.

No Hoisting Before Initialization: Unlike var, let variables are not accessible before they are declared (temporal dead zone).

3. const Scope

Block Scope: Like let, const is block-scoped.

Constant Value: const is used to declare variables that cannot be reassigned (though their contents may still be mutable if they are objects or arrays).

No Hoisting Before Initialization: Similar to let, const variables are not accessible before declaration (temporal dead zone).

2. Tell us the use cases of null and undefined

1. A variable is undefined when it is declared but has not been assigned a value, or when a function does not explicitly return a value.

The undefined value can be

  • Uninitialized Variables
  • Function Parameters Without Arguments
  • Functions Without a Return Statement
  • Accessing Non-Existent Object Properties
  • Array Elements with "Holes"

2. Null is an intentional assignment to indicate the absence of any object or value.

3. What do you mean by REST API?

A REST API (Representational State Transfer Application Programming Interface) is a widely used architectural style for designing networked applications. It allows systems to communicate over the internet using a set of standard principles and protocols, primarily HTTP. REST APIs enable developers to build web services that are lightweight, scalable, and maintainable.
